Quick Start¶

Update Documentation¶

Clone project¶

git clone https://github.com/FinTrak-Solutions/Backend.git

Install virtual env¶

cd Backend
virtualenv venv
source venv/bin/activate
pip3 install mkdocs

Goto dir¶

cd Documentation

Modify .md in docs/¶

# check, update locally
mkdocs serve
# modify .md in docs/

Build and deploy¶

# build and deploy
mkdocs build
mkdocs gh-deploy
