API¶

    • Json<Vec<String>>: Empty

    • STATUS_CODE: OK (200)
    • Json<Vec<String>>: Json<["Account Summary:", "td_debit: -$3200", "Category Summary: ", "food: -$200", "clothes: -$300"]>
